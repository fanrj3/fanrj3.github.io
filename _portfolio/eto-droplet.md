## Overview

A 3D interactive simulator based on WebGL technology that recreates the iconic "Droplet" weapon from Liu Cixin's *The Three-Body Problem*. Players control an advanced alien probe tasked with destroying human spacecraft fleets. Built as a course project for Computer Graphics.

## Features

- High-speed physics simulation (reaching ~1% light speed)
- Volumetric particle explosion effects
- Intelligent AI fleet mechanics
- Cinematic camera system
- Immersive audio system
- Multiple input support: keyboard, mouse, and game controller
- Cross-platform: web application and standalone desktop software

## Tech Stack

- **3D Rendering**: Three.js (r181)
- **Build Tool**: Vite 7.2
- **Desktop Framework**: Electron 39.2
- **Packaging**: Electron Builder 26.0
